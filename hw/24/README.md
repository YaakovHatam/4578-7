# Nodejs basic app

## Server
### packages
#### Dependencies
- `express`
- `cors`
#### Dev dependencies
- `nodemon`

### Create files
Create your first files: 
- `index.js`
- `dogs.js`

### Endpoints
#### Get `/dog/`
send back random dog image source

## Client
simple `index.html` file and `jquery` that fetch random image from the server endpoint and show the image in html page
